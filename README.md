# Keysh

Manage your SSH keys and identities with ease!

## Installation

Add this line to your application's Gemfile:

    gem 'keysh'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install keysh

## Usage

run `keysh help` for help!
